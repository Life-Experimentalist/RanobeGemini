## Interaction Patterns

### Sliders
- **Temperature & Font Size**:
  - Display updates in real-time as you drag
  - Notification appears when you release (change event)
  - Value persists in browser storage

### Toggles
- **Theme, Analytics, Auto-Hold**:
  - State changes immediately
  - Saved to storage on change
  - Notification confirms action

### Dropdowns
- **Theme Mode**:
  - Changes apply immediately
  - Notification confirms selection
  - Syncs with popup settings

### Full-Width Section
- **Auto-Add by Site**:
  - Expands to full modal width
  - Each site configured independently
  - Scrollable if many sites

## Design Principles Applied

1. **Visual Hierarchy**: Emoji icons + color coding guide users
2. **Information Architecture**: Related settings grouped by function
3. **Space Efficiency**: 4-column bento layout maximizes space
4. **Progressive Disclosure**: Full-width auto-add moves below
5. **Responsive Design**: Gracefully adapts to all screen sizes
6. **Accessibility**: Color + text labels, no color-only indicators
7. **Consistency**: Matches popup Advanced tab sections
