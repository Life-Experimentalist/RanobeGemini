Frequently Asked Questions

How much does the Gemini API cost?
The Gemini API offers a free tier with generous usage limits that are sufficient for personal use. You don't need to provide payment information to get started.
Is my API key secure?
Yes, your API key is stored only in your browser's local storage and is only sent directly to Google's servers when processing content. We never see or store your API key on our servers.
What happens if I don't provide an API key?
The extension requires a valid Gemini API key to function. Without one, you won't be able to enhance novel translations.
Can I change my API key later?
Yes, you can update your API key at any time through the extension's popup interface.
