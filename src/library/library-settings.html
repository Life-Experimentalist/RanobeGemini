🎨 Content Boxes

Define custom CSS classes for content regions. Gemini will be instructed to wrap matching content with your class names.

ℹ️ How it works
Each box type defines a CSS class (e.g. rg-my-box) and a hint for Gemini (e.g. "stat windows, pop-ups"). When you enhance a chapter, Gemini will wrap matching content in <div class="rg-my-box">…</div>. The CSS is automatically injected into every page you visit. See the Content Styles Showcase for a live preview of all built-in and custom box types.
