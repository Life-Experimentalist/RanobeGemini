# Ranobe Gemini Icons

This folder contains various icon sizes needed for this extension:

- `icon.png` - Small toolbar and favicon icon (16×16)
- `icon.png` - Windows high-DPI favicon (32×32)
- `icon.png` - Extension management page icon (48×48)
- `logo-96.png` - Extension detail page icon (96×96)
- `logo-128.png` - Extension store icon (128×128)
- `logo-256.png` - High-resolution devices (256×256)
- `logo-512.png` - Marketing and large displays (512×512)
- `logo-1024.png` - Original source files (1024×1024)

## Naming Convention
- Small icons: `icon.png` - Used for small images
- Large logos: `logo-{size}.png` - Used for larger more detailed images
