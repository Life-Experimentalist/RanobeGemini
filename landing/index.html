How it works

From install to first summary in minutes

  1. Install

    Load the Edge or Firefox build from the latest release. Chromium variants are coming; Safari is not planned.

  2. Pick a chapter

    Open a supported site. The extension detects chapters and adds inline controls.

  3. Enhance or summarize

    Click Enhance for polished text or Summarize for quick context. Long chapters are chunked safely.

  4. Export anytime

    Export your library to JSON for safekeeping and re-import later. No cloud dependency.

FAQ

Answers at a glance

üí° What is Ranobe Gemini?

Ranobe Gemini is a browser extension that uses Google's Gemini AI to enhance machine-translated web novels, improving readability, grammar, and flow while maintaining the original story integrity. It also provides summaries and site-aware content processing.

üîë Do I need an API key?

Yes. Add your free Gemini API key from Google AI Studio in the popup Settings tab. Backup keys and rotation are supported for rate limits. The free tier is generous and suitable for most users.

üåê Which sites are supported?

Currently supports: Archive of Our Own (AO3), FanFiction.net (desktop + mobile), Ranobes, ScribbleHub. WebNovel is temporarily disabled while we refine it. More sites may be added based on user requests.

üíæ How does content caching work?

Enhanced chapters are automatically cached for 7 days. When you revisit a chapter, the cached version loads instantly. You can regenerate fresh content or delete the cache using the buttons in the enhanced banner.

üìö How do I import or export?

Popup ‚Üí Library tab ‚Üí Export downloads a JSON; Import restores it later. No cloud dependency. You can also use the backup features in the Advanced tab for automatic backup rotation.

üîÑ How do I toggle original vs enhanced?

After enhancement, a banner appears at the top letting you switch between original and enhanced, or delete cached copies. Use the Show Original or Show Enhanced buttons.

‚öôÔ∏è Which model should I pick?

Gemini 2.0 Flash is balanced and recommended for most users; 2.5 Flash is fastest for quick reads; 2.5 Pro is highest quality with larger context window but slower.

üîß What if enhancement fails?

Test your key in the popup (Settings ‚Üí Test Key), enable chunking for long chapters under Advanced settings, lower temperature to 0.3, and retry. Check console (F12) for detailed errors. Report persistent issues on GitHub.

üé® What do advanced parameters do?

Temperature controls creativity (lower = consistent, higher = creative). Top-P limits token diversity. Top-K sets maximum tokens per step. Max Output sets response length. Use defaults unless you understand these.

üìã What are Custom Prompts?

Custom prompts let you control how Gemini processes content. Enhancement Prompt improves translation quality, Summarization Prompt creates summaries, Permanent Prompt adds global instructions. Reset to defaults if unsure.

üí∞ Is it free?

Yes! The extension is open-source and free. You only pay for Gemini API usage (free tier is generous). Check Google AI pricing for rate limits.

üìñ How does the Library work?

The Library tab tracks all novels you've read, showing current novel details, suggestions, and complete reading history organized by website. Export/import your library as JSON for backups.
