How it works

From install to first summary in minutes

  1. Install

    Load the Edge or Firefox build from the latest release. Chromium variants are coming; Safari is not planned.

  2. Pick a chapter

    Open a supported site. The extension detects chapters and adds inline controls.

  3. Enhance or summarize

    Click Enhance for polished text or Summarize for quick context. Long chapters are chunked safely.

  4. Export anytime

    Export your library to JSON for safekeeping and re-import later. No cloud dependency.

FAQ

Answers at a glance

What is Ranobe Gemini?

A browser extension that uses Gemini API to enhance and summarize web novel chapters with site-aware prompts.

Do I need an API key?

Yes. Add your Gemini API key in the popup Settings tab. Backup keys and rotation are supported for rate limits.

Which sites are supported?

AO3, FanFiction (desktop + mobile), Ranobes, ScribbleHub. WebNovel is temporarily disabled while we refine it.

How do I import or export?

Popup ‚Üí Library tab ‚Üí Export downloads a JSON; Import restores it later. No cloud dependency.

How do I toggle original vs enhanced?

After enhancement, a banner lets you switch between original and enhanced, or delete cached copies.

Which model should I pick?

Gemini 2.0 Flash is balanced; 2.5 Flash is fastest; 2.5 Pro is highest quality.

What if enhancement fails?

Test your key in the popup, enable chunking for long chapters, lower temperature, and retry. Check console for errors.

Is it free?

Yes. The extension is open-source. You only pay for Gemini API usage (free tier is generous).
