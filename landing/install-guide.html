Temporary / Side-load Install

Use these steps for Chrome, Brave, Opera, Vivaldi, Ulaa, Arc, and other Chromium browsers.

1. Get build package

Use either local package output or GitHub release zip:

  • Local: releases/ (firefox/chromium packages)
  • GitHub: Releases page artifacts

2. Load unpacked/temporary

  • Firefox: about:debugging → This Firefox → Load Temporary Add-on
  • Chromium browsers: extensions page → Developer mode → Load unpacked

3. Re-load after updates

Temporary installs may reset IDs. If ID changes, update Google OAuth redirect URI accordingly.
