Troubleshooting

Common issues

Popup still shows Disconnected

Open the popup again after completing OAuth. If it still shows disconnected, verify your redirect URI matches the extension ID and try reconnecting.

Auth failed (invalid_client)

Ensure you are using the correct Client ID and that the redirect URI is registered exactly (including /drive).

Access blocked or testing only

Add your Google account as a test user or publish the consent screen.
